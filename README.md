# 🚀 CommandHub

Proyecto sencillo que permite ejecutar scripts mediante exec(). Para ejecutar scripts más complejos o con gran volumen de datos, se recomienda usar spawn().

---

## ✨ Características Principales

* **Funcionalidad A:** Crea y guarda script orginazados en tags.
* **Funcionalidad B:** Ejecución de esos script con exec.
* **Optimización:** Pequeña validicion para evitar scrips peligrosos(Se puede mejorar).

## 🛠️ Tecnologías Utilizadas

| Tecnología | Uso Principal |
| :--- | :--- |
| **Lenguaje/Entorno** | JavaScript / TypeScript (Node.js) |
| **Framework/Tareas** | NestJS con child_process |
| **Base de Datos** | PostgresSQL |

## 🚀 Instalación y Uso

Para ejecutar este proyecto localmente, sigue estos pasos:

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/lautaroalfonso423/web-defender.git

2. **Instala dependencias:**
    ```bash
   npm install

2. **Inicia el proyecto:**
    ```bash
   npm run start:dev
